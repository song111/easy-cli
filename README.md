## 开发

开发合并到develop，发布合并到master分支发布
```bash
clone https://github.com/song111/easy-cli.git 

cd easy-cli 

# 切换到develop分支开发
git checkout develop

# 安装
yarn 

# 编译所有文件，并监听变化
npm run dev

# 打包所有文件
npm run build

```
